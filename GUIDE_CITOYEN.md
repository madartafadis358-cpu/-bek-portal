## 1. Accéder au portail

**Lien :** [https://bek-portal.onrender.com](https://bek-portal.onrender.com)

Le site s'ouvre sur n'importe quel appareil : PC, tablette, smartphone (Android/iPhone).

**Navigation :**
- **Barre du haut** (navbar) : tous les espaces du site (Accueil, Signalements, Propositions, Projets, etc.)
- **Menu burger (≡)** sur mobile : affiche la navigation complète
- **Boutons "Connexion" / "Rejoindre"** : en haut à droite

**Vidéo suggestion 1** : *"Comment naviguer sur le portail BEK"* (30s) — montrer l'ouverture du site, la navigation desktop et mobile.

## 2. Signaler un problème

**📌 But :** Déclarer un souci dans la commune (voirie, éclairage, propreté, sécurité, etc.)

1. Cliquez sur **"Signalements"** dans la barre de navigation
2. Cliquez sur **"Nouveau signalement"**
3. Remplissez :
   - **Titre** (ex : "Nid de poule rue des Fleurs")
   - **Catégorie** (Voirie, Éclairage, Propreté, etc.)
   - **Description** (détail du problème)
   - **Quartier** (où se situe le problème)
   - Ajoutez une **photo** (optionnel mais recommandé)
4. Cliquez **"Publier"**

Votre signalement est visible par tous les citoyens et les administrateurs.

**Vidéo suggestion 2** : *"Comment signaler un problème en 30 secondes"* (40s) — montrer la création d'un signalement avec photo.

## 3. Proposer une idée

**💡 But :** Suggérer une amélioration pour la commune

1. Cliquez sur **"Propositions"** dans la navigation
2. Cliquez sur **"Nouvelle proposition"**
3. Remplissez :
   - **Titre** (ex : "Installer des bancs publics")
   - **Catégorie**
   - **Description** (expliquez votre idée en détail)
4. Cliquez **"Publier"**

Les autres citoyens peuvent voter 👍 pour soutenir votre idée. Les plus populaires remontent en haut de la liste.

**Vidéo suggestion 3** : *"Proposer une idée et voter"* (35s) — montrer la création et le vote.

## 4. Participer à un projet

**🚀 But :** Rejoindre un projet citoyen en cours

1. Cliquez sur **"Projets"** dans la navigation
2. Parcourez les projets actifs (parc municipal, bibliothèque, formation numérique...)
3. Cliquez sur un projet pour voir les détails :
   - Avancement (pourcentage)
   - Nombre de bénévoles
   - Description complète
4. Contactez l'équipe via les informations fournies

**Vidéo suggestion 4** : *"Découvrir et rejoindre un projet citoyen"* (30s) — navigation dans la liste des projets.

## 5. Offrir ou recevoir de l'aide

**🤝 But :** Proposer son aide (bénévolat, expertise) ou demander de l'aide

### Pour offrir :
1. Cliquez sur **"Entraide"**
2. Cliquez sur **"Publier une annonce"**
3. Choisissez le type : Bénévolat, Aide matérielle, Expertise professionnelle
4. Décrivez ce que vous proposez
5. Indiquez un contact (email ou téléphone)

### Pour demander :
1. Parcourez les annonces disponibles
2. Contactez directement la personne via les informations fournies

**Vidéo suggestion 5** : *"Publier une annonce d'entraide"* (30s) — création d'une annonce de bénévolat.

## 6. Soutenir financièrement

**❤️ But :** Faire un don pour soutenir le portail

1. Cliquez sur **"Soutenir"** dans la navigation
2. Entrez le montant en DZD (minimum 50 دج)
3. Optionnel : votre nom et un message
4. Cliquez sur **"Soutenir"**
5. Paiement sécurisé via **Chargily Pay** (CIB / Edahabia)
6. Après paiement, un reçu s'affiche

**Vidéo suggestion 6** : *"Faire un don en 1 minute"* (45s) — montrer le formulaire et le paiement Chargily.

## 7. S'inscrire comme membre

**👥 But :** Devenir membre officiel de la communauté BEK

1. Cliquez sur **"Rejoindre"** en haut à droite
2. Remplissez :
   - **Nom complet**
   - **Email** (optionnel)
3. Cliquez **"S'inscrire"**
4. Votre inscription est soumise à validation par un administrateur

Une fois validé, vous apparaissez dans le compteur de membres et pouvez accéder à votre tableau de bord personnel.

**Vidéo suggestion 7** : *"Créer son compte citoyen"* (25s) — formulaire d'inscription simple.

## 8. Publier une annonce publicitaire

**📢 But :** Faire la publicité de votre commerce ou activité (payant)

1. Cliquez sur **"Publicité"** dans la navigation
2. Choisissez votre formule :
   - **Sidebar** (barre latérale droite) → 500 DZD/mois
   - **Header** (en-tête du site) → 1000 DZD/mois
   - **Inline** (dans le fil d'actualité) → 300 DZD/mois
3. Remplissez :
   - **Titre** de l'annonce
   - **Image** (format paysage recommandé)
   - **Lien** vers votre site ou page
4. Payez via Chargily Pay
5. L'annonce est activée après confirmation du paiement

**Vidéo suggestion 8** : *"Créer une annonce publicitaire"* (50s) — montrer les 3 formules et le paiement.

## 9. Questions fréquentes

**❓ Le site est gratuit ?**
Oui, pour les citoyens (signaler, proposer, voter). Seules les annonces publicitaires sont payantes.

**❓ Qui gère le site ?**
Le Super Administrateur **Brik Chaouche Mourad** et son équipe.

**❓ Comment contacter l'équipe ?**
- Page **"Contact"** sur le site
- Email : b.mourad.computer@gmail.com
- Téléphone : 0771 26 37 48

**❓ Mes données sont-elles protégées ?**
Oui. Les mots de passe sont cryptés (bcrypt), les connexions sont sécurisées (JWT), et le site utilise HTTPS.

**❓ Le site ne répond pas ?**
Le site peut mettre 30 secondes à se réveiller après une période d'inactivité. Rafraîchissez la page et attendez quelques secondes.
